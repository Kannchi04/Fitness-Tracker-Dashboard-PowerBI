# Fitness Tracker Dashboard - Power BI

## Project Overview

This project is an interactive Power BI dashboard focused on fitness, wellness, nutrition, sleep, and workout analysis. The dashboard provides insights into user workout patterns, calorie intake, mood tracking, heart rate trends, and overall health metrics.

The dashboard was built using Power BI with multiple pages to analyze different aspects of fitness and wellness data.

---

## Key Files

* Fitness tracker.pbix : Main Power BI dashboard file
* workout_fitness_tracker_data.xlsx : Dataset used for analysis

---

## Dashboard Pages

### 1. Summary Dashboard

Provides a high-level overview of fitness metrics including:

* Average resting heart rate
* Daily calorie intake
* Sleep hours
* Body fat percentage
* Workout duration
* Mood distribution after workout

### 2. Workout Overview

Analyzes:

* Workout duration by workout type
* Workout intensity trends
* Calories burned vs workout duration
* Workout distribution across weight ranges

### 3. Calories and Nutrition

Tracks:

* Average calories burned
* Water intake
* Daily calorie intake
* Body fat percentage
* Nutrition trends across workout types and genders

### 4. Sleep and Mood Tracker

Analyzes:

* Sleep duration trends
* Mood before and after workouts
* Relationship between sleep and mood
* User distribution by gender and mood

### 5. Heart Wellness Insights

Provides insights into:

* Average heart rate trends
* Heart rate distribution across workout intensity
* Heart rate vs step count
* Sleep impact on heart rate
* Heart wellness trends by age and gender
